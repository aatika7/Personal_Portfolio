## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone [your-repo-link]
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
